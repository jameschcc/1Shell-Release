# 1Shell -- Modern SSH Client for Desktop

**1Shell** is a modern, cross-platform SSH terminal client built with Flutter. It is fast, beautiful, and designed for developers and sysadmins who manage multiple servers.

---

## Features

| Capability | Description |
|---|---|
| SSH / Telnet / Serial | Connect via SSH, serial port, or Telnet |
| X11 Forwarding | Seamless X11 forwarding for remote GUI apps |
| Zmodem Transfer | Upload/download files over SSH with rz/sz |
| Multi-Tab + Groups | Organize connections with folders |
| Multi Auth | Password, public key, jump host/bastion |
| GPU Accelerated | Smooth terminal with GPU rendering |
| Command Palette | Quick access to all features |
| Local Terminal | Works without a remote connection |

## Download and Install

**Debian / Ubuntu:**
```bash
sudo dpkg -i 1shell_1.11.17_amd64.deb
```

**Fedora / RHEL:**
```bash
sudo rpm -ivh 1shell_1.11.17_amd64.rpm
```

- .deb (15.7 MB): https://github.com/jameschcc/1Shell-Release/releases/download/v1.11.17/1shell_1.11.17_amd64.deb
- .rpm (19.5 MB): https://github.com/jameschcc/1Shell-Release/releases/download/v1.11.17/1shell_1.11.17_amd64.rpm

## Why 1Shell?

- vs Termius: Fully offline, data stays on your machine, no subscription
- vs SecureCRT: Modern UI, cross-platform, free to use
- vs PuTTY: Tab management, session save, modern terminal
- vs MobaXterm: Lightweight, cross-platform (not Windows-only)
